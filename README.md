# 🎙️ SaarthiAI – Voice-Enabled Assistive Platform for Inclusive Digital Access

## 🔍 Overview
SaarthiAI is an AI-powered voice-enabled assistive platform designed to make digital systems accessible to everyone, including visually impaired users, elderly individuals, and people with low digital literacy. The platform enables users to interact with digital services such as government portals, healthcare systems, and educational platforms using simple voice commands and guided navigation.

The project focuses on bridging the digital divide by providing an inclusive and user-friendly interface that reduces dependency on complex visual interactions.

---

## 🚀 Key Features
- 🎤 Voice-controlled navigation for hands-free interaction  
- 🔊 Text-to-Speech (TTS) assistance for better accessibility  
- 🌐 Multilingual support (Tamil, English, Hindi)  
- 👁️ Designed for visually impaired users  
- 👵 Simple and intuitive interface for elderly users  
- ⚡ Fast and efficient interaction with digital platforms  

---

## 🛠️ Tech Stack
- Programming Language: Python  
- Technologies: Speech Recognition, Text-to-Speech (TTS), Natural Language Processing (NLP)  
- Frontend: HTML, CSS, JavaScript  

---

## 🧠 System Workflow
1. User gives input through voice commands  
2. Speech Recognition converts voice to text  
3. NLP processes the command and identifies intent  
4. System performs the requested action  
5. Response is delivered using Text-to-Speech

## ⚙️ Installation & Setup

1. Clone the repository  
git clone https://github.com/yourusername/saarthiai-voice-assistive-platform  

2. Navigate to project directory  
cd saarthiai-voice-assistive-platform  

3. Install dependencies  
pip install -r requirements.txt  

4. Run the application  
python app.py
